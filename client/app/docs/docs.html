Researcher Documentation


We built the scaffold, now you build the building


What You Need:

A parallelized algorithm or MapReduce that we can run on our cluster. We have not yet automated the process of submitting projects, so if you are interested, contact us at projects.crowdcrunch@gmail.com


What We Provide:

What you will need to do is pick a framework that runs on mesos--or make your own--and write a scheduler for your algorithm or computation.

We will verify and security-check your framework and then integrate it into the crowd crunch platform. The project will be featured on the website and in the OSX application.
